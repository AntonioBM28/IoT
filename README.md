# Personaje 2024
## Nombre del personaje
Muñeco de Nieve con un árbol de navidad
## Creador
Balderas Melchor Jose Antonio
## Explicación de funcionamiento
El muñeco tendra la funcionalidad de encender luces led, tener movimiento y contar con algunos parlantes y el árbol encenderan sus luces
